# MoodLog
A quiet corner on your macOS desktop to log how you feel — 365-day mood heatmap and a note to your future self.

## Status
🚧 In active development — not yet available as a standalone app.

## Development
Built with vibe coding using Claude as an AI pair programmer.

## Planned Features
- [x] 365-day mood heatmap
- [x] Mood keyword selection (positive / negative)
- [x] Score-based color intensity
- [ ] Settings (theme color, language)
- [ ] Year-end reflection
- [ ] Annual archive
- [ ] macOS menu bar widget
- [ ] Auto-launch on login
- [ ] Export as .dmg
